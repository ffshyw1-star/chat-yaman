## 📦 التثبيت

### المتطلبات
- Node.js 14+ 
- npm أو yarn

### خطوات التثبيت

1. استنساخ المستودع:
```bash
git clone https://github.com/ffshyw1-star/chat-yaman.git
cd chat-yaman
```

2. تثبيت الحزم:
```bash
npm install
```

3. إنشاء ملف `.env`:
```bash
cp .env.example .env
```

4. تشغيل التطبيق:
```bash
npm start
```

5. فتح المتصفح:
```
http://localhost:3000
```

## 🚀 البدء السريع

### للتطوير
```bash
npm run dev
```

يستخدم nodemon لإعادة التحميل التلقائي.
